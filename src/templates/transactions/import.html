CSV Format Requirements

Required Columns

Your CSV file must contain the following columns in this exact order:

  1. Date - Transaction date (YYYY-MM-DD, MM/DD/YYYY, or DD/MM/YYYY)
  2. Description - Transaction description
  3. Amount - Transaction amount (positive number)
  4. Type - Transaction type (income, expense, or transfer)

Optional Columns

You can also include these optional columns:

  • Payee - Who you paid or received money from
  • Category - Transaction category
  • Notes - Additional notes

Example CSV Format

Date,Description,Amount,Type,Payee,Category,Notes
2024-01-15,Grocery Shopping,85.50,expense,Walmart,groceries,Weekly shopping
2024-01-16,Salary Payment,3500.00,income,ABC Company,salary,Monthly salary
2024-01-17,Gas Station,45.20,expense,Shell,transportation,

Important Notes

  • All amounts should be positive numbers (the type determines if it's income or expense)
  • Dates can be in various formats, but YYYY-MM-DD is recommended
  • Transaction types must be: income, expense, or transfer
  • Categories should match existing categories or will be set to "other"
  • Empty cells are allowed for optional columns
  • File size limit is 10MB
