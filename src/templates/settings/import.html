Import Guidelines (Future)

Supported File Formats

  • CSV (Comma Separated Values)
  • Excel (.xlsx, .xls)
  • Text files (.txt)
  • JSON format

Data Requirements

  • Date column (required)
  • Amount column (required)
  • Description or memo (recommended)
  • Category (optional)
  • Account information (optional)

Best Practices

  • Ensure data is clean and properly formatted
  • Remove any header or footer information
  • Use consistent date formats
  • Backup existing data before importing
  • Review imported data for accuracy
